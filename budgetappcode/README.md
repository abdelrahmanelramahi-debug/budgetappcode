# Finance Command

Finance Command is a lightweight, single-page finance app for budget tracking and
daily spending control. It runs entirely in the browser and stores its state in
localStorage.

## What it does
- Budget plan builder with categories and items
- Daily entry ledger for quick spend/adjust actions
- Weekly allowance tracking with rollover limits
- Food cycle tracker with buffer/lock tracking
- Surplus/deficit indicator with a resolution flow
- Liquidity breakdown and reality check reconciliation
- Global undo for recent actions

## How it works
- Open `newmain.html` in a browser
- Data persists locally; no server required

## Tech
- Vanilla HTML, CSS, and JavaScript
- Tailwind CDN for utility styling
